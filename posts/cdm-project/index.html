The CDM Project

The CDM Project was a set of tools and guides used for testing Widevine DRM on multiple websites & platforms. I used it to learn how Widevine works and how I can avoid getting my content easily downloaded.

The cdm-project.com website has been down for a few months now and I don’t think it’s coming back up anytime soon. I’ll be posting the backup I have. I hope it doesn’t get taken down. I won’t be hosting any of the files myself so it should be a grey area.
